What is a DOM?

Not as scary as it sounds! DOM stands for Document Object Model.

DOM shows the structure of your HTML document (nice looking trees!). It usualy follows indentations and depicts relationships (parent-child nodes).

DOM is very useful because it allows you to access elements of a website quicker! For an example, try var parent = document.childnodes[1];.


What are Block and In-Line Elements?

Do not despair! The answer is actually quite simple. Check it out:

BLOCK ELEMENTS IN-LINE ELEMENTS
These normally start with a new line when displayed in a browser. These are normally displayed without starting a new line.
I'm talking h1, p, ul, etc. I'm referring to b, td, a, etc.
A main example of a block element is div. DIV is used as a container for grouping other elements. The browser will display a line break before and after it. A main example of an in-line element is span. SPAN is used as a container for text.
With CSS, it can also be used to set style attributes to large blocks of content. With CSS, it can also be used to set style attributes to parts of text.

What are ID and Class Selectors?

Easy peachy. See below:

ID SELECTOR CLASS SELECTOR
ID selectors are used to specify a style for a SINGLE, unique element. Class selectors are used to specify a style for a GROUP of elements.
You'll recognize it in the CSS file by "#". You'll recognize it in the CSS file by ".".
